# Calories Burnt Predictions  
Using data from https://www.kaggle.com/datasets/ruchikakumbhar/calories-burnt-prediction

Applying the models on fictitious, competition data and values to predict CE for: https://www.kaggle.com/competitions/playground-series-s5e5/data

Exploring the data and various ML models for predicting calorie expenditure during workout.

## Project as package
The pyproject.toml is configured in package format, so that all reusable python modules are subfolders of `./src`.  
The package also needs to be installed (once) in editable format with `uv pip install -e .`
